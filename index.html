About

I'm a web developer in central PA specializing in hand-coded HTML, CSS, and JavaScript. I work primarily with client-side optimization, making sure your site looks great in every browser and runs well on every device.

Whether you're building an Angular app or updating a ten year old CRM system, I can get you where you need to be. There's always a way to make things work, and I can help you find the cleanest, most efficient way to do it.

Demos

These are personal projects that I dip into from time to time. More current professional examples can be provided upon request via the Contact section.

Click the images below to view a live demo, or click the GitHub link to view the source code.


Starfield

 

An HTML/CSS/jQuery animation of stars passing by, built to act as a lightweight background or sidebar animation. It's responsive, browser-agnostic, doesn't use images, and can be easily embedded almost anywhere.

3D Business Card

 

An HTML/CSS/jQuery proof-of-concept for a rotating virtual business card. I used a combination of folding and rotating animations to create the card itself as well as the flipping animations. I owe a big thanks to the awesome JavaScript library transit.js, which made my code much more concise and legible, while keeping the animations cross-browser compatible and mobile-friendly using a single method.

SpaceTrade

 

A small JavaScript game similar to Dope Wars and other resource management games, but also includes a randomized encounter system, a Choose-Your-Own-Adventure mechanic, and some experimental UI elements. It also features the Starfield background referenced elsewhere on this page, running continuously behind the game (quietly and efficiently).

It was a refreshing change to develop this, and I learned (and re-learned) a lot about object/variable manipulation in the process, as well as organizational best practices. I rarely update this demo unless I find an addition that I think is particularly clever, but I frequently re-use modules of this project in other larger, sometimes commercial projects.
